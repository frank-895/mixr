# Mixr

This app uses Vite, React, TypeScript, and Convex.

## Local checks

- `npm run format` formats the repo with Biome.
- `npm run lint` runs Biome's linter.
- `npm run check` runs Biome's formatter and linter checks together.
- `npm run typecheck` runs `tsc -b`.
- `npm run build` creates a production build with Vite.
